# 你好,我是 Qing Li 👋

我构建**自托管基础设施与全栈开发者工具**——主要使用 Go、Python 和 TypeScript,也涉足 CUDA、固件和 Android 设备移植。

## 我在做的方向

- 🧠 **LLM 基础设施** —— API 网关、多 Agent 流水线编排、可观测性
- 📡 **自托管服务** —— 通知中枢、流媒体、家庭实验室工具
- ⚡ **系统与并行计算** —— CUDA 核函数、MPI/OpenMP、OpenWrt 固件、AOSP 设备树
